# 📊 Sales Analysis Dashboard (Tableau)

## 📌 Overview
This project presents an interactive Tableau dashboard for analyzing sales performance.

The dashboard provides a clear view of key metrics, helping to understand trends, compare regions, and evaluate overall business performance.

## 🛠 Tools & Technologies

- Data Visualization
- Dashboard Design
